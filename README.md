# ✨ 待办清单

一个功能丰富、界面精美的前端待办清单应用，纯 HTML/CSS/JS 实现，无需构建工具，浏览器直接打开即可使用。

## 🚀 快速开始

1. 下载 `index.html`
2. 用浏览器打开，即刻使用

## ✨ 功能特性

### 核心功能
- **添加任务**：输入任务内容，一键添加
- **完成/取消完成**：点击复选框标记任务状态
- **删除任务**：支持删除单个任务
- **编辑任务**：点击任务文字即可编辑

### 任务属性
- **优先级**：支持高（🔴）、中（🟡）、低（🟢）三级优先级
- **分类标签**：个人（🏠）、工作（💼）、学习（📚）、其他（🎯）
- **截止日期**：可设置任务截止日期

### 视图与筛选
- **全部 / 进行中 / 已完成**：Tab 快速切换视图
- **分类筛选**：按标签筛选任务
- **搜索**：🔍 关键词搜索任务内容
- **进度追踪**：进度条展示完成率（完成数 / 总数）

### 数据管理
- **导出**：支持导出为 JSON 或 Excel（📥）
- **导入**：支持导入 JSON 文件恢复任务（📤）
- **清空已完成**：一键清理已完成任务

### 个性化
- **主题切换**：内置多款主题（💜 橹穆·紫、💙 橹穆·蓝、💜 橹穆·靛）
- **自定义主题**：支持创建个人主题
- **自动轮播主题**：定时自动切换主题
- **自定义背景**：支持上传本地图片作为背景

## 🛠 技术栈

- HTML5 + CSS3 + 原生 JavaScript
- 零依赖（导出 Excel 使用 SheetJS CDN）
- localStorage 本地持久化存储
- 响应式设计，支持移动端
